# NBA MVP Predictor 
A machine learning project that predicts the NBA Most Valuable Player (MVP) winners from using historical player statistics and team performance data from 1980 to 2025.
## Overview 
NBA MVP voting is influenced by a combination of individual statistical dominance, team success, and narrative factors. This project builds a Random Forest Classifier to identify MVP winners based purely on box score and advanced statistics, then analyzes where and why the model succeeds or falls short.
